## Installation
Create [.env](./.env) file in the root folder. That contain environment variable need to be configured:

```
ACCESS_TOKEN=<your-access-token>
```

Install all required packages

```shell
pip install -r requirements.txt
```
## Usage
Run with:

```shell
python lab1/main.py
```
